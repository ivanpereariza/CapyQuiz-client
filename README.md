# Client routes

## Index

| URL            | Description       | Protected |
| -------------- | ----------------- | --------- |
| `/`            | Index page        |           |
| `/profile/:id` | User profile page | ✅       |
| `/signup`      | Signup page       |           |
| `/login`       | Login page        |           |
| `/ranking`     | Ranking page      |           |


## Quizzes

base URL `/quizzes`

| URL            | Description       | Protected |
| -------------- | ----------------- | --------- |
| `/`            | Quizzes list page |           |
| `/details/:id` | Quiz details page |           |
| `/edit/:id`    | Quiz edit page    | ✅       |
| `/play/:id`    | Quiz play page    | ✅       |
| `/create`      | Create new Quiz   | ✅       |
